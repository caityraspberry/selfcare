Welcome Project

On the first day of Girls Who Code, we were taught the basics of computer science using Scratch. After being introduced to an entirely new aspect of technology, we were asked to create a short, little project using Scratch to display our interests and teach our peers about ourselves. This is the first project I ever coded!

Draw Shapes with Petals

After being introduced to loops and variables we were asked to make a program that drew different shapes in Scratch. This specific program has two different aspects to it. The first part asks the user the number of sides they would like their polygon to have. The second part asks the user how many petals they want. After that, the script draws that polygon the requested amount of times rotating a certain angle each time to make it appear like a flower with petals.

Feel free to check out my project here

Paddle Ball

After working with Scratch we were asked to code a simple paddle ball game consisting of two users. We were taught conditionals which we were able to use while making this game. We had to set restrictions so the ball would actually bounce and we had to use conditionals in order to make the game have multiple rounds and a way to keep score. The most difficult part of this project was learning about angles to get the ball to move in the way we wanted.

Draw Shapes

In Scratch, we used loops, functions and angle measurement to design a program that draws simple shapes. After the program asks the user how many sides they want it calculates the measurement of its interior angles. Once it does that the program is ready to draw a shape knowing how many times it repeats the loop and how it should turn.

Shapes in Shapes

This is another program that uses Scratch to draw shapes except it has a small twist. As oppose to asking the user how many sides they would like, they ask how long they want the sides to be. The program then draws eight shapes with a different number of sides but the same side length to create a very fancy shape.

Scratch Game

Our final project in Scratch was to create a game. This was without a doubt the most challenging and the most rewarding Scratch project we did. My specific game has a bird who is falling and is in need of the player to help them avoid obstacles. It has two different modes, day and night. We used the random control to make it so objects appear in random places at random times to make it interesting. It has a score component as well and it takes your high score.
